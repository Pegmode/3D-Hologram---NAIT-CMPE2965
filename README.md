# 3D Hologram Capstone Project - CMPE2965 NAIT
Capstone project for Computer Engineering Technology at the Northern Alberta Institute of Technology.<br>


by: Kaden Sparrow and [Daniel Chu](http://pegmode.github.io)<br>
course: CMPE2965-A02-1252

# Synopsis
This project is a rotating 3D volumetric display. Comprised of a rotating 16x32 LED display on a custom 6 layer circuit board with integrated microcontroller. Controlled via control UI running on windows. Able to load 3D models in the `.obj` format.
