# Server Tools
This folder contains server side custom tools
- **rotationProof**: Animated visual prover for board dimensions and rotations parameter.
- **voxelLookupGenerator**: Tool to generate programming language agnostic lookup table cartesian to cylinrical coordinates.
- **voxelConversion**: Simple conversion toolkit for converting `.vl32` 3D voxel files into usable (x,y,z) coordinates for python. Includes a simple matplotlib visualiser.
